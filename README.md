# Bella Nails - Aplicación de Manicura

Aplicación web profesional para salón de manicura con sistema de gestión completo.

## Características

- Panel administrativo para gestión de trabajadoras y configuraciones
- Galería de trabajos realizados
- Sistema de citas y reservas
- Configuración personalizable de marca y colores
- Interfaz responsive y moderna

## Tecnologías

- React + TypeScript + Vite
- Supabase (Backend y autenticación)
- Tailwind CSS + shadcn/ui
- React Query para gestión de estado
